## Main Message

Security Review Workflow is an AI-assisted DevSecOps workflow that reviews GitHub pull request changes, detects risky infrastructure and CI/CD settings, notifies Slack, and creates remediation pull requests after human approval.

The project intentionally uses L2 approval-based operation as the default safety model. It does not directly apply production changes without human review.
